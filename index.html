1. Overview

This Privacy Policy explains how our Instagram Post Verification Application ("the App") collects, uses, and protects your information when you use our service to verify your Instagram account and access your Instagram posts.

2. Information We Collect

2.1 User-Provided Information

  • Instagram username
  • Verification codes sent through Instagram Direct Messages
  • Profile information associated with your Instagram account

2.2 Instagram Post Data

  • Post content and media URLs
  • Captions and descriptions
  • Post timestamps
  • Engagement metrics (likes, comments)
  • Location data (if included in posts)

2.3 Technical Information

  • Device information
  • App usage data
  • Error logs and performance data

3. How We Use Your Information

3.1 Account Verification

  • Generating unique verification codes
  • Processing verification requests through Instagram DMs
  • Validating Instagram usernames and account ownership

3.2 Post Management

  • Retrieving and displaying your Instagram posts
  • Storing post data for faster access
  • Maintaining post engagement metrics

3.3 Service Improvement

  • Analyzing app performance
  • Debugging and error resolution
  • Enhancing user experience

4. Data Storage and Security

4.1 Storage Infrastructure

We use Supabase, a secure database service, to store:

  • Verification codes and their status
  • Instagram post data
  • User verification status

4.2 Security Measures

  • Encrypted data storage
  • Secure API endpoints
  • Regular security audits
  • Access control and authentication

5. Instagram API Usage

We interact with Instagram's services through:

  • Meta's Graph API for message handling
  • Instagram Business Account integration
  • Webhook implementations for real-time updates

6. Data Retention

6.1 Verification Data

  • Verification codes are stored temporarily
  • Codes expire after a set period
  • Verified status is maintained until account removal

6.2 Post Data

  • Instagram posts are cached for improved performance
  • Post data is updated periodically
  • Historical post data is maintained until user requests deletion

7. User Rights and Controls

You have the right to:

  • Access your stored data
  • Request data deletion
  • Revoke app access to your Instagram account
  • Update or correct your information
  • Opt-out of data collection

8. Third-Party Services

We utilize the following services:

  • Supabase for data storage
  • Meta's Instagram Graph API
  • Instagram Business Platform

9. Changes to This Policy

We may update this privacy policy to reflect:

  • Changes in our practices
  • New features or services
  • Legal or regulatory requirements

Users will be notified of significant changes through the app or email.
